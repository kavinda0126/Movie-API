# Movie REST API

This is a RESTful API for managing movies and a watchlist using Node.js, Express, and MongoDB.

## Features

- **Movies**
  - View all movies
  - View a single movie by ID
  - Add a new movie
  - Update a movie
  - Delete a movie

- **Watchlist**
  - View watchlist
  - Add a movie to the watchlist
  - Remove a movie from the watchlist
